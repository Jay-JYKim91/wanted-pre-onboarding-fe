## 원티드 프리온보딩 프론트엔드 - 선발 과제

원티드 프리온보딩 프론트엔드 선발 과제로 제공해주신 API를 이용한 Todo App입니다.
`React` `TypeScript` `react-router-dom` `axios` `react-query` `Material UI`

### 실행 방법

```
yarn install
yarn start
```

`localhost:3000` 접속
